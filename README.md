# 美食博客生成器

灵感来源于A岛的 [一个帖子](https://adnmb2.com/t/24736659)。

## 运行方法

首先，克隆本仓库。

```shell
git clone https://github.com/luochang212/FoodBlogGenerator.git
```

接着，在命令行打开 `FoodBlogGenerator` 文件夹，然后运行：

```shell
python 美食博客生成器.py
```

最后，输入文章要讨论的主题，比如茄子。
